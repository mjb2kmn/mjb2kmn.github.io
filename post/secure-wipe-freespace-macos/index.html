Secure Wipe Freespace on macOS
2017-08-13

Why

After deleting a file with sensitive info in it, or after editing sensitive info out of a file, the data still exists on disk until it’s overwritten.

Here is how you can overwrite all free space on your disk, or even wipe all space on a disk requiring it to be repartitioned and formatted.

How

Use diskutil command with the secureErase option. Mulitple methods are available, run diskutil secureErase to see them. In this example I will be writing 0x00 bytes. Be sure to use your acutal volume name if it’s different from the example.

!!! Warning !!!

Do not forget to include freespace option as the default is the wipe the entire volume.

sudo diskutil secureErase freespace 0 /Volumes/Macintosh\ HD

diskutil can also be used to wipe an entire external drive

sudo diskutil secureErase 0 /dev/disk2

Tested on macOS Sierra - 10.12.x.
